A versatile 4-channel VCA mixer module for Eurorack format synthesizers.

== Overview

The Quad VCA Mixer combines four voltage-controlled amplifiers with a mixer in a compact Eurorack module. Each channel features independent CV control and manual level adjustment.

== Features

* 4 independent VCA channels
* Per-channel CV control
* Mix output
* Compact 3-board design
